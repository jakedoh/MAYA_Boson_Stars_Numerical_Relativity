
Expanding Universe Project

Implementation of the FLRW metric as background for black hole simulations with moving puncture gauge conditions.

Folders: 
   - BSSN : Implementation of the BSSN equations (with and without matter)
   - MyThornsGRHydro : Thorns for the matter evolution, initial and Boundary conditions based on the hydro code MyThornsGRHydro
   - MyThornsIllinois : Thorns for the matter evolution, initial and Boundary conditions based on the hydro code IllinoisGRMHD
   - Parfiles : Parfiles for runs with and without black hole
   - Thornlists : Thornlists for both versions (GRHydro and IllinoisGRMHD)
